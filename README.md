# Spotify Music Analytics Dashboard (2010–2019)

This project is an interactive Power BI dashboard that analyzes Spotify music trends from 2010 to 2019. It explores how genres, audio features, artists, and songs influence streaming performance and popularity.

The dashboard provides insights into:
1. Most streamed genres
2. Top artists and songs
3. Audio feature patterns (danceability, energy, valence, etc.)
4. Streaming trends over time

Key Insights
1. Pop and Dance Pop dominate streaming volume, indicating strong preference for mainstream commercial music.
2. Streaming trends show a steady increase over time with fluctuations across years.
3. Genres with higher danceability tend to achieve higher streams, suggesting users prefer rhythmic music.
4. Energy alone is not strongly correlated with popularity, showing that multiple factors influence success.
5. Spotify consumption is concentrated in a few genres, showing a long-tail distribution of music demand.

Tools 
1. Power BI Desktop
